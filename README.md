# API Tester Frontend

This is the **frontend** of the API Tester application, built with **Next.js** and **React**, deployed on **Vercel**. It allows users to test APIs by sending requests and viewing responses in a simple interface.

## Live Demo

Frontend Application: [https://api-tester-app-ochre.vercel.app](https://api-tester-app-ochre.vercel.app)  
Backend API: [https://api-tester-backend-3qmj.onrender.com](https://api-tester-backend-3qmj.onrender.com)

## Features

- Send **GET**, **POST**, **PUT**, and **DELETE** requests to any API endpoint.
- View response **status codes**, **headers**, and **response body**.
- User-friendly interface with **responsive design**.
- Configurable **backend URL** via environment variables.

## Tech Stack

- **Frontend:** React, Next.js  
- **Styling:** Tailwind CSS   
- **Hosting:** Vercel

## Usage

- Enter the API endpoint you want to test.
- Select the request method (GET, POST, PUT, DELETE).
- Add headers or body if required.
- Click Send to view the response.
